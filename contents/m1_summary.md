- **时间**：2024.9 - 2025.1
- **状态**：课程学习 / 科研入门 / 实习准备

### 📚 课程成绩
> **GPA：3.92** &nbsp;|&nbsp; **平均分：94.18**

- **信息系统安全：98**
  - 考核为期末考试80%与平常两次作业汇报（量子计算与鸿蒙系统安全漏洞）。
  - 考试开卷，很友好，考试题目会涉及两次作业相关内容（记得打印），记得去图书馆借本书。
  - 书上内容比PPT更详细，直接抄就完事。

- **信息系统集成技术：98**
  - 六个作业，作业形式类似软件工程汇报，需要做PPT，用Visio画图，Word写报告。
  - 给分很好，平常作业都是97、98，也算掌握一些作图写报告技能。

- **物联网安全：95**
  - 考核占比未知，给分友好。
  - 随机签到2次，来的人有加分，没来不扣分。
  - 考核形式为汇报+报告，选择一个和物联网安全相关的进行汇报，注意汇报脱稿和时间都要准备很好。

- **新中特：94**
  - 推荐选王娜娜老师课程，考核平常雨课堂刷课+签到占比50%（往年40%），仅多送10分。
  - 考试开卷，今年大事件：二十届四中全会。材料里面会有一些涉及到，大部分同学带了书、PPT和辅导百问（还是很有用的）。

- **矩阵理论与应用：93**
  - 线性代数的进阶，考试占比70%，平时成绩白送30%。
  - 大量往年题参考，考试里面有很多往年题目类似的。考试错了个判断（不会）和两个填空（粗心，不变因子和满秩分解）。

- **网络空间安全数学基础：92**
  - 比较全面讲述了网安研究生需要的一些数学知识。
  - 内容包含：图论 / 格 / 线性代数 / 概率论 / 群环域 / 基础数论 / 组合数学等。
  - 考试占比70%，考试题目比较简单，均为计算题，有往年题参考基本上十拿九稳。
  - 平常作业+小组汇报和个人心得占比30%。

- **机器学习：91**
  - 考核分为平常作业和期末考试。
  - 平常作业占比50%，分为个人与团队作业。在Kaggle平台进行编程测试，以最后提交文件的准确率排名和报告撰写给分。
  - 期末考试参考往年6系+神经网络大模型相关基础知识。

- **科学写作与报告：PASS**
  - 助教古法点名，前8周老师上课讲写作与论文相关知识，后8周分组汇报（按格式和要求，分别汇报小组和个人）。
  - 成绩通过制度，不计入GPA。

- **英语阅读/写作/口语：EX**
  - 开学后英语六级达到550或者通过免修考试者，可以英语免修，成绩记为EX，获得2学分。
  - 成绩不计入GPA的计算，考试分为笔试和口语两部分，分别为70分和30分。
  - **笔试**：选择题，题型类似雅思，有雅思真题（木塔），选题目/阅读/判断对错/摘要句子等。
  - **口语**：分为两部分，读文章（150s）和给一个Topic进行阐述（120s）。
